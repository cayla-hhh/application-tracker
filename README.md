<h1>Job Application Tracker</h1>

A web-based dashboard to help users organize, track, and manage their job applications in one place.

<h2>Features</h2>

User Authentication: Secure login system to keep your data private. 🔑

Job Management: Add new applications with titles, companies, deadlines, and statuses. ➕

Dynamic Sorting: Organize your list by Company Name, Status, or Deadline. 📊

Full CRUD: Edit existing entries to update progress or delete them when finished. 📝

Clean UI: A professional, responsive dashboard styled with custom CSS. 🎨

<h2> Technologies Used </h2>

Backend: Python with the Flask web framework. 

Database: SQLite3 for local data storage. 

Frontend: HTML5 and CSS3 (utilizing Flexbox for layout).

<h2>Installation & Setup</h2>

Initialize the Database: Run the setup script to create the tables and a test user.

    python main.py

Run the App: Start the Flask server.

    python app.py

Access the Tracker: Open your browser and go to http://127.0.0.1:5000/.
